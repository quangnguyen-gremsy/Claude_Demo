## Data Model (Output Schema)

```json
{
  "image": "frame_001.jpg",
  "backend": "pytorch",
  "benchmark": {
    "runs": 100,
    "mean_ms": 18.4,
    "std_ms": 1.2,
    "min_ms": 16.9,
    "max_ms": 23.1,
    "fps": 54.3
  },
  "detections": [
    {
      "label": "person",
      "confidence": 0.92,
      "bbox": [120, 85, 340, 420]
    }
  ],
  "total_objects": 1
}
```

## Implementation Steps

**Step 1 — CLI & Input Validation**
- Parse args: `image_path`, `--backend` (pytorch|onnx), `--runs` (default 100)
- Validate image file exists
- Validate backend value

**Step 2 — Image Preprocessing (shared)**
- Load image via OpenCV
- Resize to 640x640, normalize to [0,1]
- Convert to tensor (PyTorch) hoặc numpy array (ONNX)
- Preprocessing phải giống nhau cho cả 2 backends để kết quả so sánh công bằng

**Step 3 — Backend: PyTorch**
- `torch.hub.load('ultralytics/yolov5', 'yolov5s', pretrained=True)`
- Set `model.conf = 0.4`
- Inference: `results = model(image_path)`
- Extract detections từ `results.pandas().xyxy[0]`

**Step 4 — Backend: ONNX**
- Load model: `onnxruntime.InferenceSession('yolov5s.onnx')`
- Input: preprocessed numpy array, shape `[1, 3, 640, 640]`, dtype float32
- Run: `session.run(None, {input_name: input_array})`
- Post-process: NMS + confidence filter thủ công (ONNX không tự làm)

**Step 5 — Benchmark Loop**
- Warm up: chạy 5 lần, bỏ kết quả (tránh JIT / cache cold start)
- Measure: chạy `--runs` lần, ghi `time.perf_counter()` mỗi lần
- Tính: mean, std, min, max (đơn vị ms), fps = 1000 / mean_ms

**Step 6 — Save & Print**
- Write `benchmark.json`
- Print summary:
  ```
  Backend : pytorch
  Runs    : 100
  Mean    : 18.4 ms  |  FPS: 54.3
  Std     : 1.2 ms
  Min/Max : 16.9 / 23.1 ms
  Objects : 3 (person: 0.92, car: 0.87, ...)
  ```

## Assumptions

1. ONNX model (`yolov5s.onnx`) đã được export sẵn — script không tự export
2. Cả 2 backends dùng cùng input size 640x640
3. ONNX session chạy trên CPU (không handle GPU provider)
4. Benchmark đo inference time only — không tính preprocessing time

## Edge Cases cần test

| Case | Expected Behavior |
|------|------------------|
| Image file không tồn tại | Exit rõ ràng, không crash |
| `--backend onnx` nhưng file `.onnx` không có | Thông báo rõ, gợi ý export command |
| Không detect được object nào | `detections: []`, `total_objects: 0` |
| `--runs 0` hoặc âm | Validate và reject ngay |
| ONNX vs PyTorch output khác nhau | Đây là expected (NMS khác nhau) — cần document |
